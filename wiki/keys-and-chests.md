# Keys And Chests

## Key Tiers

| Key | Opens |
| --- | --- |
| Bronze Key | Bronze chest |
| Silver Key | Silver chest |
| Gold Key | Gold chest |

Keys go directly into the player's key ring instead of taking inventory space.

## Chest Tiers

| Chest | Typical Reward Shape |
| --- | --- |
| Bronze | Small gold burst and one modest reward |
| Silver | Better gold, a stronger gear roll, and support loot |
| Gold | Premium gold and multiple high-value rewards |

## Key Sources

- Ground pickups
- Merchant stock
- Elite / boss-related drops
- Route-choice guaranteed arrivals

## Boss Chest Flow

- Boss chest starts locked
- Boss dies
- Matching key drops from the keeper
- Chest unlocks
- Player opens the prompt and confirms the haul

This keeps boss rewards dramatic without making the normal key economy irrelevant.
